O que são pré-processadores?

Pré processadores são interpretadores de código, onde uma sintaxe é convertida para outra sintaxe, (SCSS para CSS), por exemplo.

  • É possível usar recursos que ainda não existem no CSS, como: Variáveis, nesting, mixins, herança e etc.
  • Facilita a manutenção do código, tornando o código mais limpo.
  • Faz o código CSS mais flexível e reutilizável.
  • AJuda evitar a duplicação de código.
